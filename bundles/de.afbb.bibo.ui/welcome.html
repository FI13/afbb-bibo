Allgemein

Einrichtung

Als erstes sollten sie das Standard-Passwort ändern und für sich und andere Verwalter jeweils einen neuen Benutzer anlegen.

Als nächstes sollten sie die verschiedenen Kategorien von Medien erstellen, die Sie erfassen wollen. Sie können jeweils einen Namen und ein Icon pro Kategorie festlegen.

Anschließend können sie mit der Erfassung ihres Bestandes beginnen.

Übersicht der Oberfläche

Die Oberfläche teilt sich in 3 Hauptbestandteile auf:

  • Am oberen Rand liegt die Toolbar, auf ihr befinden sich die wichtigsten Funktionen.

  • Auf der linken Seite liegt die Navigations-Ansicht. In dieser Ansicht werden die verwaltbaren Elemente dargestellt. Um zu einer Detail-Ansicht zu gelangen, klicken sie doppelt auf einen Eintrag. Weitere Funktionalitäten sind über ein Popup-Menü zu erreichen.

  • Auf der rechten Seite liegt der Bereich für die verschiedenen Detail-Ansichten. Es kann nur jeweils eine Ansicht einer Art gleichzeitig geöffnet werden. Sollte es die Möglichkeit geben Änderungen zu speichern, so erscheint ein Disketten-Symbol in der rechten oberen Ecke. Wenn dieses ausgegraut ist, so ist zur Zeit ein Speichern nicht möglich oder nötig. Zwischen den einzelnen Ansichten kann mit den entsprechenden Reitern gewechselt werden.

Übersicht der Funktionen

Hier können sie ihr eigenes Passwort ändern.
Hier können sie einen neuen Verwalter für die Software einrichten.
Hier können sie einen neuen Ausleiher hinterlegen. Als Ausleiher werden Personen bezeichnet, die später Medien ausgeliehen bekommen sollen.
Hier können sie neue Kategorien von Medien erfassen. Jedes Medium ist einer Kategorie zugeordnet und das ausgewählte Logo wird in der Navigationsansicht angezeigt.
Hier können sie neue Medien erfassen. Scannen sie dazu zuerst jeweils den Barcode des Mediums ein und anschließend die ISBN (für Medien, die keine ISBN haben, können sie hier auch eine Seriennummer eintragen) - es sollten sich im Idealfall alle weiteren Felder füllen. Sie können nun die weiteren Felder bearbeiten und eventuell noch einen Zustand des Mediums angeben. Felder, die mit einem * markiert sind, sind Pflichtfelder und müssen ausgefüllt werden um fortfahren zu können.
Hier können sie Medien an einen Ausleiher verleihen. Dieses Element ist normalerweise deaktiviert, es sei denn sie haben in der Navigationsansicht einen Ausleiher markiert, dem sie Medien verleihen wollen. Sie erreichen diese Funktionalität auch, wenn sie einen Rechtsklick auf den Ausleiher machen und den gleichen Menüpunkt wählen. Wenn sie den Hacken bei Liste drucken nicht entfernen, so wird bei Abschluss eine Übersichtsliste der ausgeliehenen Medien gedruckt.
Hier können sie Medien zurücknehmen. Sie haben bei jedem Medium die Möglichkeit den Zustand zu bearbeiten.
Hier können sie eine Inventur ihrer Medien vornehmen. Bei dieser Inventur wird der Zeitpunkt der letzten Inventur für jedes inventarisierte Medium auf das Tagesdatum gesetzt. Sie haben außerdem die Möglichkeit, den Zustand der Medien anzupassen.
